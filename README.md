# Math 280 - Code Repository (UC Merced)

Welcome to the **Math 280** code repository at UC Merced! This repository hosts **code and relevant data** for the course.

> **Note:** Lecture notes and other course materials are available on **Canvas**. This GitHub repository is focused on **examples, exercises, and data for computational work**.

---

---

### `Examples/`

Code examples organized by week:

**Week 3 — Linear Regression**
- `3-ESL-ProstateCancer-LinearRegression.ipynb` — Adapted from [ESL Python Notebooks](https://github.com/empathy87/The-Elements-of-Statistical-Learning-Python-Notebooks)
- `3-ISLR-LinearRegression.ipynb` — Adapted from [ISLR Python](https://github.com/collinprather/ISLR-Python)

**Week 4 — Interpolation**
- `Cha1_PolyInterpolation_Bishop.ipynb` — Polynomial interpolation example from Bishop

**Week 7 — Classification**
- `iris_classification.ipynb` — Classification methods using the Iris dataset

**Week 9 — Deep Learning & Double Descent**
- `Discussion_ISLR_Chapter10_MNIST_Lite.ipynb` — MNIST walkthrough (lite version)
- `Example1_GaussianLinearModel_DoubleDescent.ipynb` — Double descent in Gaussian linear models
- `Example2_ISLR_Chapter10_SplineApproximation_DoubleDescent.ipynb` — Double descent via spline approximation

Additional examples and exercises will be added throughout the semester.
